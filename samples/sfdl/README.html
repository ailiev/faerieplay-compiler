SFDL sample programs

The source files have .sfdl extensions. Some of them have matching .cjs files with sample input data. (cjs is Cson, see README).

Preparation

Build and install the Faerieplay system. Most of it is needed, except the MIPS emulator and its RAM plugins.

Example: gcd program

$ cd BASEDIR/sfdl
$ make gcd.runtime

# in another terminal:
$ card_server -d /tmp/faerieplay

$ cvm gcd.runtime < gcd-input.cjs
Check out the other SFDL (ls *.sfdl) example files in the same manner.
