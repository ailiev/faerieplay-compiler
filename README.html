Faerieplay Compiler

This is sfdlc, the Faerieplay compiler, which translates a high level language into an arithmetic circuit extended with gates for accessing indirect arrays.

Texinfo manual

The sfdlc documenation is maintained in texinfo form, and built with 
$ cd sfdl-compiler
$ make doc
Then, the manual is available in html form, and also as an info file: 
$ cd sfdl-compiler
$ info -f doc/sfdlc.info

Example programs

Several example SFDL programs are provided in BASEDIR/sfdl/.

The compiler has test programs in BASEDIR/sfdl-compiler/tests/.
