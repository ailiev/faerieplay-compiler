Faerieplay Compiler

This is sfdlc, the Faerieplay compiler, which translates a high level language into an arithmetic circuit extended with gates for accessing indirect arrays.

Languages

Sfdl&mash;secure function definition language—is similar to C with a bit of Pascal sprinkled in. Sfdlc also has support for a subset of C++, which is not fully tested yet.

Using

Please consult the info file, doc/sfdlc.info, which is built as part of building the compiler, eg: 
$ info -f doc/sfdlc.info

Example programs

Several example programs are provided in BASEDIR/sfdl/.

The compiler has test programs in tests/.
