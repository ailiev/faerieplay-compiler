Faerieplay Compiler

This is sfdlc, the Faerieplay compiler, which translates a high level language into an arithmetic circuit extended with gates for accessing indirect arrays.

Source Language Syntax Reference

The syntax reference documents are mentioned in the Texinfo manual, but here we can provide direct links to them. They are generated with the BNF converter:
SFDL: Doc.pdf
FC++: Doc.pdf

Texinfo manual

The sfdlc documenation is maintained in texinfo form. It is available in html form and as an info file: 
$ cd sfdl-compiler
$ info -f doc/sfdlc.info
(After the documentation has been built.)

Example programs

Several example SFDL programs are provided in BASEDIR/sfdl/.

The compiler has test programs in BASEDIR/sfdl-compiler/tests/.
