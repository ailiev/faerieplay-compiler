Faerieplay Compiler

This is sfdlc, the Faerieplay compiler, which translates a high level language into an arithmetic circuit extended with gates for accessing indirect arrays.

Please see also the introduction to the compiler in the manual (available after make doc is run).

Languages

Sfdl—secure function definition language—is similar to C with a bit of Pascal sprinkled in. Sfdlc also has support for a subset of C++, which is not fully tested yet. Both languages are documented in the info manual.

Using

Please consult the manual (which is built as part of building the compiler).

Also, an info file is built, doc/sfdlc.info: 

$ info -f doc/sfdlc.info

Example programs

Several example programs are provided in BASEDIR/sfdl/.

The compiler has test programs in tests/.
